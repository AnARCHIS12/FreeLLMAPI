FreeLLMAPI is a self-hosted gateway for free-tier LLM providers. Add keys, route requests, track usage and expose a simple API for your apps.

Install locally Production deploy

Unified provider gateway

11 providers · encrypted keys · automatic fallback

What it does

Simple enough for beginners, useful enough for builders.

No cloud account required to start locally.
01

Unified API

One `/api/chat.php` endpoint routes requests to the configured provider pool.

02

Encrypted keys

Provider keys are stored locally and encrypted before being saved.

03

Fallback routing

If a provider is unavailable or limited, FreeLLMAPI can try the next route.

Beginner install

Double-click or run one file.

Launchers can install PHP automatically when possible.
1

Download the project

Put the folder anywhere on your computer.

2

Start it

Windows: `start.bat`, macOS: `start.command`, Linux: `./start.sh`.

3

Add keys

Open the Admin page and paste your provider keys.

local install
# Windows
> start.bat

# macOS
$ ./start.command

# Linux
$ ./start.sh

# Open in browser
http://localhost:3001

Production with domain

Docker + Caddy + automatic HTTPS.

Point your domain to the server IP, set `DOMAIN=your-domain.com`, then run the production launcher.

See command
production deploy
# Linux/macOS VPS
$ ./start-prod.sh

# Windows with Docker Desktop
> start-prod.bat

# Then edit .env.prod
DOMAIN=your-domain.com

# Run the same launcher again
https://your-domain.com
